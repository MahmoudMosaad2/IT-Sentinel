# دليل تحويل البرنامج إلى تطبيق EXE مع تثبيت واجهة سطح المكتب 🚀

مرحباً بك! لقد قمت ببرمجة وتطبيق ميزة **الفحص النشط** وميزة **التحقق من حسابات مسؤولي النظام (Domain Admin Verification)**. 

الآن، لتشغيل هذا المشروع كبرنامج محلي بالكامل على نظام التشغيل Windows بصيغة (`.exe`) يقوم تلقائياً بإنشاء أيقونة على سطح المكتب وفتح المتصفح فور الضغط عليه، إليك الطريقة الأتوماتيكية والاحترافية خطوة بخطوة.

لقد قومت بإعداد الكود داخل مشروعك لكي يدعم التشغيل الذاتي وفتح المتصفح تلقائياً عند الإطلاق. لتجميع البرنامج في ملف `.exe` واحد، استخدم إحدى الطرق التالية:

---

## 💡 الطريقة الأولى: استخدام سكربت التشغيل الصامت الصغير (الأخف والأسرع ⚡)

هذه الطريقة ممتازة جداً، حيث تقوم بإنشاء ملف تشغيل صغير لا يظهر شاشات سوداء مزعجة، ويقوم بتشغيل السيرفر وفتح المتصفح فوراً، ويمكنك تحويله لملف `.exe` بضغطة زر واحدة!

### الخطوة 1: إنشاء ملف التشغيل الصامت
عند تحميلك لمشروعك بصيغة **ZIP** وفك الضغط عنه على جهاز الكمبيوتر الخاص بك، قم بإنشاء ملف نصي جديد وقم بتسميته `Run-IT-Sentinel.vbs` في المجلد الرئيسي للمشروع، واكتب بداخله الكود التالي:

```vbs
Set WshShell = CreateObject("WScript.Shell")
' تشغيل السيرفر في الخلفية بدون شاشة سوداء
WshShell.Run "cmd.exe /c npm run start", 0, False
```

### الخطوة 2: إنشاء أيقونة واختصار على سطح المكتب
لعمل أيقونة احترافية على سطح المكتب ترسلها لأي مستخدم آخر:
1. اضغط بـ الزر الأيمن للماوس على ملف `Run-IT-Sentinel.vbs` واختر **Send to -> Desktop (create shortcut)**.
2. اذهب إلى سطح المكتب، واضغط كليك يمين على الاختصار الذي تم إنشاؤه واختر **Properties** (خصائص).
3. اضغط على زر **Change Icon** ثم اختر أي أيقونة درع أو حماية تفضلها من ملفات النظام أو أي ملف `.ico`.
4. أعد تسمية الاختصار إلى اسم برنامجك المفضل (مثال: `IT-Sentinel Admin Scanner`).

*بمجرد نقر أي مستخدم عليه، سيفتح التطبيق في الخلفية ويفتح المتصفح الافتراضي له تلقائياً على واجهة تسجيل الدخول المؤمنة!*

---

## 🛠️ الطريقة الثانية: تجميع المشروع بأكمله داخل ملف EXE واحد مستقل (Portable EXE)

إذا كنت ترغب في دمج بيئة Node.js بالكامل مع الكود في ملف تفيذي واحد (`.exe`) لتشغيله على أي جهاز كمبيوتر بدون الحاجة لتثبيت أي برامج مسبقة:

### الخطوة 1: تثبيت حزمة التجميع المخصصة (pkg)
افتح التيرمنال (Command Prompt) في مجلد مشروعك بعد تحميله ثم اكتب الأمر التالي لتثبيت أداة التجميع:
```bash
npm install -g pkg
```

### الخطوة 2: تجميع المشروع لقاعدة بيانات ويندوز
قم بتشغيل بناء ملفات الصفحات أولاً:
```bash
npm run build
```

ثم قم بتشغيل أمر التجميع مباشرة لإنتاج ملف الـ EXE المتوافق مع أنظمة Windows الـ 64-bit:
```bash
pkg dist/server.cjs --targets node18-win-x64 --output IT-Sentinel.exe
```

الآن سيظهر لديك ملف ممتاز ومستقل باسم `IT-Sentinel.exe` بمجرد نقره سيقوم بـ:
1. إطلاق خادم الويب الخفي خلف الكواليس.
2. استرداد وفحص الأجهزة الحية على النطاق.
3. فتح المتصفح الافتراضي للمستخدم تلقائياً والتوجه نحو صفحة فحص الـ Admin!

---

## 🔧 أين تم كتابة هذه الأكواد داخل هذا المشروع؟
لقد قمت بدمجها وترتيبها لك في المواقع التالية لكي تعمل بدون أي تدخل منك عند التصدير:
1. **في ملف `server.ts` (السطر 364 وما بعده):** أضفت كوداً ذكياً يتحقق إذا كان البرنامج يعمل محلياً خارج الحاوية السحابية (على نظام Windows الخاص بك)، ويقوم بتشغيل أمر فتح المتصفح الافتراضي للجهاز تلقائياً لعنوان `http://localhost:3000` بعد الاستعداد الفوري للسيرفر.
2. **في واجهة تسجيل الدخول `index.html`:** تم تحديث نظام تسجيل الدخول لكي يحتوي على زر **"فحص الحساب"** لتأكيد تبعية المدخلات وصحتها من Active Directory والـ OU المعتمد قبل السماح بنقر زر "دخول النظام".
